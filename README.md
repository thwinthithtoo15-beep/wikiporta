# 🌐 Dokuwikiporta
**The Next Generation of Decentralized Knowledge Base.** *Inspired by DokuWiki and built for Digital Freedom.*

---

## 🚀 Overview
**Dokuwikiporta** is a lightweight, zero-backend, and portable wiki system. It is designed to be the "Successor of Wikipedia" for the modern age, focusing on user ownership and ease of use. 

Built by **Thwin (a 13-year-old student from Myanmar)**, this project aims to provide a "Lightweight Alternative" to DokuWiki for those who cannot or do not want to set up a PHP server.

## ✨ Key Features
- 📂 **File-Based:** Automatically fetches and renders `.md`, `.txt`, and `.html` files.
- 🚀 **Zero Server Required:** Runs entirely in the browser using JavaScript. No PHP or Database needed.
- 📱 **Fully Portable:** A single HTML file that you can carry in your USB stick.
- 🎨 **Modern & Minimalist:** Uses `water.css` and `marked.js` for a clean, professional look.
- 🛡️ **Privacy Focused:** No ads, no tracking, and no profit-driven features.

## 🛠️ Tech Stack
- **HTML5 / CSS3**
- **JavaScript (ES6)**
- **Libraries:** - [marked.js](https://marked.js.org/) (Markdown Parser)
  - [water.css](https://watercss.kognise.dev/) (Lightweight Styling)

## 📖 How to Use
1. Clone or download `index.html`.
2. Place your content files (`home.md`, `about.txt`, etc.) in the same folder.
3. Open `index.html` via a local server (like Live Server or Python SimpleHTTPServer) to avoid CORS issues.
4. To add pages, update the `Navigation` section in the HTML or add to the `wikiData` object.

## 📜 Manifesto
1. **Knowledge belongs to the people**, not corporations.
2. **Data Ownership:** You should be able to copy your "house" and move it anywhere.
3. **No Gatekeepers:** Accessing information should be simple and free.
