## 🚀 Overview

Wikiporta is a modern, fast, and structured wiki engine designed to manage information using ****JSON**-based data architecture**. Unlike traditional wikis, Wikiporta separates content from metadata, allowing for powerful data visualization and cross-referencing.

## ✨ Key Features

- **Structured Data:** Built-in support for **JSON** metadata (similar to Wikidata).
- **Fast Rendering:** Minimalistic **CSS**/JS stack for lightning-fast performance.
- **Markdown Support:** Write content in easy-to-read Markdown.
- **Enterprise Security:** Protected by **Cloudflare **WAF**** and Geo-blocking technology.
- **Legal Protection:** Officially certified under the **Berne Convention** (Registration No: CR-**2026**-**16352**).

## 🛡️ Security & Access Control

To protect intellectual property, Wikiporta implements a strict security policy:
- **Geo-blocking:** Access is restricted in regions lacking Berne Convention compliance (IR, IQ, SO, AO, ET).
- **Anti-**VPN**/Proxy:** Hardened by **WAF** rules to prevent unauthorized content scraping.
- **Blockchain Anchoring:** Data integrity is verified on the **Polygon Blockchain**.

## 🛠️ Tech Stack

- **Frontend:** **HTML5**, **CSS3** (Inter Font Family), JavaScript.
- **Styling:** Bootstrap 5.3 (Clean UI).
- **Markdown Engine:** Marked.js.
- **Data Format:** **JSON**.
